## 📌 Description

Astrofix Note is an application for creating and managing notes.  


## 🚀 Features

- Create and edit notes  
- Save and manage files  
- Work with folders  
- Hotkey support  

## 🛠 Installation

```bash
git clone https://github.com/alt-enterssx/astrofix-note.git
cd astrofix-note

mkdir build
cd build
qmake ../astrofix.pro
make
./AstrofixNote
